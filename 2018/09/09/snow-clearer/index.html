铲雪车

9月 9, 2018

输入格式

输入数据的第1行表示铲雪车的停放坐标（x,y），x，y为整数，单位为米。下面最多有100行，每行给出了一条街道的起点坐标和终点坐标，所有街道都是笔直的，且都是双向一个车道。铲雪车可以在任意交叉口、或任何街道的末尾任意转向，包括转U型弯。铲雪车铲雪时前进速度为20 km/h，不铲雪时前进速度为50 km/h。

保证：铲雪车从起点一定可以到达任何街道。

输出格式

铲掉所有街道上的雪并且返回出发点的最短时间，精确到分钟。

Analysis

这题很无聊啊，车道是双向的，又要把雪除尽，肯定要一条路走两次，又说所有的街道都可以到达，变相的告诉我们图是联通的，你连连通性都不用判断，直接算出输入数据街道的长度，全部*2，就是总路程长度。

这题的输出比较坑，要输出小时+分钟，小时直接把小数向下取整，分钟*60，小数四舍五入是我新学会的，(n+0.5)向下取整。

Code O(n)
