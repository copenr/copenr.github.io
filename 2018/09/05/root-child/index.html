找树的根和孩子

Problem

问题描述

　　给定一棵树，输出树的根root，孩子最多的结点max以及它的孩子。

Analysis

数据太弱，直接用vector记录每个节点的儿子，然后枚举找一下根和孩子最多的节点，再输出孩子。
