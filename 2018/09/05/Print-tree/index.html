二叉树输出

9月 5, 2018


　　一棵二叉树的一个结点用一个字母表示（无重复），输出时从根结点开始：

每行输出若干个结点字符（相同字符的个数等于该结点长度），

如果该结点有左子树就递归输出左子树；

如果该结点有右子树就递归输出右子树。

假定一棵二叉树一个结点用一个字符描述，现在给出先序和中序遍历的字符串，用树的凹入表示法输出该二叉树。

Analysis

先通过先序和中序遍历把树建好。

然后再通过递归求出每一个节点的长度。

最后按照先序遍历输出。

程序大体相同。
