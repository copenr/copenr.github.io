单词查找树

9月 5, 2018

（1）根结点不包含字母，除根结点外每一个结点都仅包含一个大写英文字母；

（2）从根结点到某一结点，路径上经过的字母依次连起来所构成的字母序列，称为该结点对应的单词。单词列表中的每个单词，都是该单词查找树某个结点所对应的单词；

（3）在满足上述条件上，该单词查找树的结点数最少。

（4）样例中的单词列表就对应于提示中单词查找树。

要求：对一个确定的单词列表，请统计对应的单词查找树的结点数（包含根结点）。

Analysis

这题最开始想到的方法是按层枚举，即先看第一层需要多少个节点，再看第二层……

但是这样很麻烦，每次还要记录上自己属于哪一个分支。

之后想了想，这里直接用树的性质解决一下，先把单词排个序，然后直接找和上一个单词有多少位不一样即可。因为已经排过序了，所以肯定是相关性最大的在一起。

Code O(n*len)
