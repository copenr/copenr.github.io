Test2:第一次的风暴

Problem

问题描述

在百货公司，可以买到各种各样的正方形的地砖，为了美观起见，Randy不希望同样颜色的正方形地砖相邻。所以他找到了Tio来帮忙解决这件事情。

Tio希望在地上按照长宽划分成网格后，逐行逐列每一块的颜色组成的序列的字典序最小。可以认为，输出的方案去掉回车后形成的字符串字典序最小。

样例输入

4 3

样例输出

AAA

AAA

AAA

BCB

Analysis

一开始是想找规律填表，后来觉得规律好多，感觉还不如直接模拟。

只要认识到字典序一定是从前往后判断，要让前面的越小越好。

然后就可以直接逐行逐列填表。

我的方法比较奇怪，因为要保证是正方形砖，所以每次如果填入的字母和前几个一样，就强行把一块正方形赋值。所以有可能访问到之后的点，这样就要求每一次填字母先判断这里有没有被填过。

上午写由于思路不清晰，填字母不是从A开始枚举，而是直接填ABC，导致没考虑填D之后的情况，只有75分。。。这告诉我们不要妄下定义，我一开始认为ABC足够填。

Wrong Code

修改后程序比较清晰，而且先前直接填字母，不太好判断有没有填过，之后改用数字，0就表示没有填。
